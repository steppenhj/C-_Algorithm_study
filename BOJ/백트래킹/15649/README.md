중복되지 않게 수열 뽑는 문제이다. 
1~N 까지 M크기 수열을 뽑으면 된다.
범위가 8이하라서 아주 작다. 
int arr[10];
bool is_used[10];을 선언하고 풀 수 있다.

void func(int k)로 함수를만든다.
함수 가장 앞부분에 k==m 처리를 해서 숫자가 m만큼 되면 출력하고 줄바꿈하도록 해준다

그 아래에는 1부터 N까지 is_used==false이면 arr[k]=i로 하고 func(k+1)로 재귀호출 해준다.
is_used[i] = false를 마지막에 다시 선언해준다.

main에서는 func(0)으로 해주면 된다.
